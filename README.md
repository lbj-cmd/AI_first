#功能说明
##核心功能：
频率 - 波长计算（支持不同介质介电常数）
功率单位换算（dBm/W/dBW 互转）
传输线参数计算（VSWR / 反射系数 / 回波损耗互转）
射频常识库（核心概念 + 频段划分，标签页分类）
计算历史记录（自动保存最近 10 条，支持清空）
##界面设计：
左侧导航栏 + 右侧内容区，操作直观
支持中文显示（字体设置为SimHei）
每个功能页附带公式说明，方便学习
##使用方法：
直接运行代码（Python 内置 Tkinter，无需额外安装依赖）
输入参数后点击计算按钮，结果实时显示
##后续迭代方向
增加更多计算工具（如链路预算、滤波器参数）
优化常识库内容（增加元器件手册、标准规范）
支持结果导出（图片 / 文本）
